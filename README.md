#n-bucketize

This tool creates folders (buckets) per file's mime-type founded and move that file inside the bucket.

### Installation

```bash
npm install -g n-bucketize
```

### Usage / Bucketize

```bash
n-bucketize --dir /path/to/dir
```

### Usage / Revert

```bash
n-bucketize --dir /path/to/dir --revert
```
